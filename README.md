# Smallest, Second Smallest and Third Smallest Element Finder (C Program)

## 📌 Description
This program finds the smallest, second smallest, and third smallest elements in an array without sorting.

## 🚀 How It Works
- Assume the first element is the smallest.
- Keep track of three values:
  - Smallest
  - Second Smallest
  - Third Smallest
- Update them while traversing the array.
- Print the final results.

## 📥 Input Array

```text
50 20 80 10 5 70
```

## 📤 Output

```text
Smallest = 5
Second Smallest = 10
Third Smallest = 20
```

## 🛠️ Concepts Used
- Arrays
- Loops
- Conditional Statements
- Multiple Minimum Tracking

## 🎯 Learning Outcome
- Finding multiple minimum values
- Array traversal
- Logical problem solving
- Avoiding sorting for efficiency
